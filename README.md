# Descripción del Problema

Este proyecto aborda un problema de un juego de cartas inspirado en yu gi oh, en donde se requiere implementar el algoritmo MiniMax para que el juego se pueda jugar player vs COP

# MiniMax
El algoritmo Minimax es un método de decisión utilizado en la teoría de juegos y la inteligencia artificial para elegir el movimiento óptimo para un jugador, asumiendo que el oponente también jugará de manera óptima. Su nombre proviene de su objetivo: minimizar la pérdida máxima (o, equivalentemente, maximizar la ganancia mínima) en juegos de dos jugadores con información perfecta, como el ajedrez o las damas.
Consiste en explorar un árbol de juego, donde las capas alternan entre el turno del jugador que busca maximizar su puntaje (el "Maximizador") y el turno del oponente que busca minimizar el puntaje del primero (el "Minimizador"). El algoritmo evalúa los posibles estados del juego hasta una profundidad determinada y asigna un valor a cada estado, eligiendo finalmente la ruta que garantiza el mejor resultado para el jugador Maximizador contra el peor movimiento posible del oponente Minimizador.
